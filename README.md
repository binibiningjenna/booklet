## Features

### Core Functionality
- **Category Navigation** (Non-Fiction, Fiction, Education, Kids)
- **Book Display** with covers, details, and prices
- **Interactive Shopping Cart**:
  - Add/remove items
  - Quantity adjustment (± buttons)
  - Real-time total calculation
  - Checkout system
